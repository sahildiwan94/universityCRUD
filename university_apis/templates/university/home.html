University Apis Application


  • University Apis Application is a CRUD Api Application for Universities.

  • In addition to the CRUD Apis it consists of a Search/Filter Api for Universities.

  • The Github link to the application is Github Link.

  • The details of all the apis are given in university_apis/docs/university_apis.json file.
